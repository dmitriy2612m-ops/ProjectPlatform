# Changelog

## [1.1.0] - Текущая версия

### Изменения
- Уменьшено количество контейнеров с 8 до 6
- Удален сервис Telemetry
- Удален сервис Grafana
- Уменьшено количество реплик API с 3 до 2
- Уменьшено количество реплик Dashboard с 2 до 1

### Улучшения структуры
- Добавлен `.gitignore` для исключения ненужных файлов
- Добавлены `.dockerignore` файлы для оптимизации сборки образов
- Исправлена нумерация комментариев в `docker-stack.yml`
- Обновлены скрипты деплоя
- Обновлена документация

## [1.0.0] - Начальная версия

### Реализовано
- 6+ контейнеров (PostgreSQL, Redis, API, Dashboard, Nginx, Prometheus)
- Кластерная архитектура на Docker Swarm
- CI/CD pipeline (GitHub Actions)
- Мониторинг через Prometheus
- Балансировка нагрузки через Nginx
- Kubernetes конфигурации
